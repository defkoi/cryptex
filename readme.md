# string encryptor

## using aes-cbc and pbkdf2-sha256

## commands

- create
- set
- gen
- get
- keys

## flags

| name | default  |       hint        |
| :--: | :------: | :---------------: |
|  -f  | .cryptex |   cryptex file    |
|  -k  |          |        key        |
|  -s  |          |      string       |
|  -p  |          |     password      |
|  -i  | 600_000  |    interations    |
|  -l  |    8     | generation length |
|  -m  |   cbc    |       mode        |
